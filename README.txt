== Description ==

This add-on extends the functionality of the WordPress Swiftype Search plugin. It allows to index the custom post types, taxonomies with custom fields easily.

* Select custom post types
* Select custom fields
* Exclude by post ID
* Build custom result title
* Build custom result permalink
* Synchronize with Swiftype by latest posts


== Installation ==

1. Install the plugin in your Wordpress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Settings -> Swiftype Custom page to create your own settings.
4. Rebuild your search index by clicking the "Synchronize with Swiftype" button under Swiftype Search page.
